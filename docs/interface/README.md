# Interface Documentation

All interface specifications for this project live in this directory.

Naming rule:
- Use lowercase snake_case filenames.
- Use the suffix `_if.md` for each module specification.
- Normalize module names with underscores only; avoid spaces, `&`, and mixed case in filenames.

Current documents:
- `systolic_array_if.md`
- `mac_if.md`
- `matrix_buffer_a_b_if.md`
